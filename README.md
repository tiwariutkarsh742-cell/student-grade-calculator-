## 🚀 Project Overview
The **Student Grade Calculator** is a Python-based application that takes student marks as input and calculates grades (A–F) along with motivational messages.

This project demonstrates:
- Conditional logic (`if-elif-else`)
- Input validation using loops
- Modular programming using functions

---

## 🎯 Features
✅ Input validation (0–100 only)  
✅ While loop for error handling  
✅ Clean function-based structure  
✅ Encouraging messages for each grade  
✅ User-friendly CLI interface  

---

## 🧠 Grading Logic

| Marks Range | Grade |
|------------|------|
| 90–100     | A    |
| 80–89      | B    |
| 70–79      | C    |
| 60–69      | D    |
| 0–59       | F    |
